== Description ==

Insert screenshots in your post and pages using a simple shortcode.

* [screenshot width=200]www.wordpress.org[/screenshot]

Will show a screenshot of wordpress.org (200px width).
This plugin uses mshots from WordPress.

It's a very handy plugin, and you can use it on your portfolio page, webdirectory etc.

See screenshot examples at http://www.larsbachmann.dk/wp-screenshots

== Installation ==

1. Upload the /wp-screenshot folder to /plugins
2. Activate plugin
3. Insert shortcode to show a screenshot of a webpage
4. ex: [screenshot width=225]www.wordpress.org[/screenshot]
